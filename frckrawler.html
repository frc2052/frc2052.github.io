FRC Krawler

The fully customizable scouting solution

FRC Krawler is an Android based scouting app that was created in 2013 by Charlie, when he was a member of the FRC team 2052 KnightKrawler.

The FRC Krawler scouting system is an application that allows scouting teams to define their own metrics and create different metric categories for pit and stand scouting. These metrics only need to be created once on the main server device. Scouting can be done with multiple devices and then synced to compile the data. This data and information about a team can be stored from year to year. FRC Krawler syncs with Bluetooth, so there is no need for Internet connection. This lets teams scout and sync during the competitions and view summaries of their data collected in semi real-time. The system is also designed to be extremely reusable. Prefer an Excel format? FRC Krawler can also export to a CSV file, or Excel spreadsheets.

If you are interested in using the FRC Krawler app please follow the link to the Google Play Store to download. There is also a link to download the FRC Krawler Start Guide. Help spread the word about this FRC scouting app, FRC Krawler!

  • A simple way to keep data event-to-event, and year-to-year.
  • Simultaneous scouting for more than six team members.
  • A user interface intuitive enough for an expert, and simple enough for a beginner.
  • The ability to sync via bluetooth for use during competitions.
  • The ability to define the type of scouting you're doing, from match scouting to pit scouting.
  • Ability to search and sort based on the data collected.
  • The ability to search, and sort data based on what was collected.
  • Provides the ability to back up data onto your PC.
  • Compatability to export the data into the .csv file format.
  • Available at no cost, for anyone and everyone!
